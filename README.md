# Dotfiles

Configuration files.

## Installation

Clone the repository and run the install script:

    git clone git://github.com/7omb/dotfiles
    cd dotfiles
    ./install

Open vim and run:

    :PlugInstall

## Dependencies

- Vim 8
- Python 3 with neovim
- ag
- ctags

### Python

- jedi
- pylint

### Haskell

- stack
- ghc-mod
- hlint
- hoogle
